def modelos(P1, P2, V1, V2, id_model):
    """
    Esta funcin selecciona el modelo a utilizar basndose en el parmetro id_model y devuelve los parmetros correspondientes.
    
    Parmetros:
    P1, P2, V1, V2: Parmetros de carga
    id_model: Identificador del modelo a utilizar (0 para zapata, 1 para 1 pilote, cualquier otro valor para 2 pilotes)
    
    Devuelve:
    nodes, mats, elements, loads, UG, UC, S_nodes, E_nodes, eles: Parmetros correspondientes al modelo seleccionado
    """
       
    # Seleccin del modelo
    if id_model == 0:  # Condicin para el modelo zapata
        nodes, mats, elements, loads, UG, UC, S_nodes, E_nodes, eles = aux.zapata(P1, P2, V1, V2)
    elif id_model == 1:  # Condicin para el modelo 1 pilote
        nodes, mats, elements, loads, UG, UC, S_nodes, E_nodes, eles = aux.pilote_1(P1, P2, V1, V2)
    else:  # Condicin para el modelo 2 pilotes
        nodes, mats, elements, loads, UG, UC, S_nodes, E_nodes, eles = aux.pilote_2(P1, P2, V1, V2)
    
    # Devuelve los parmetros
    return nodes, mats, elements, loads, UG, UC, S_nodes, E_nodes, eles

  • Al evaluar con diferentes combinaciones de cargas sobre el sistema, se puede concluir que la magnitud que genera una afectacin notable sobre la cimentacin en comparacin con las otras, es el momento.
